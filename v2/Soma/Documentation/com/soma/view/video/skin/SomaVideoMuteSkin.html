Usage:
Default control skin for a SomaVideoPlayer volume mute button, automatically instantiated by the SomaVideoControls class if you dont specify one in the SomaVideoPlayer instance.

How to add controls to the player

The following code is reproducing how the default controls are added in the SomaVideoControls class (by default). 

var controls:SomaVideoControls = new SomaVideoControls();
controls.addControl(new SomaVideoPlaySkin());
controls.addControl(new SomaVideoTimeBarSkin());
controls.addControl(new SomaVideoMuteSkin());
controls.addControl(new SomaVideoFullscreenSkin());
var player:SomaVideoPlayer = new SomaVideoPlayer("video/video.flv", controls);
addChild(player);
Now you know to add controls to the player, you can build your own volume mute skin. The base will be a DisplayObject class, such as Sprite, implementing ISomaVideoMuteSkin.

Custom Mute class 
package {
    import com.soma.view.video.SomaVideoPlayer;
    import com.soma.view.video.skin.ISomaVideoMuteSkin;
    
    import flash.display.Sprite;
    
    public class CustomMuteSkin extends Sprite implements ISomaVideoMuteSkin {
        
        private var _player:SomaVideoPlayer;
        
        public function CustomMuteSkin() {
            createSkinElements();
        }
        
        private function createSkinElements():void {
            // create mute buttons here and use:
            // _player.volume = 0;
            // _player.volume = 1;
        }
        
        public function registerPlayer(player:SomaVideoPlayer):void {
            _player = player;
        }
        
        public function dispose():void {
            // remove skin elements and events listeners for cleaning when the SomaVideoPlayer instance is disposed.
        }
        
        public function muteCallBack(value:Boolean):void {
            // here the controllers will indicate when the volume is on or off.
            trace("Volume state: ", value);
        }
        
    }
}
Add the CustomMuteSkin to the SomaVideoControls instead of the default one: 
var controls:SomaVideoControls = new SomaVideoControls();
controls.addControl(new CustomMuteSkin());
var player:SomaVideoPlayer = new SomaVideoPlayer("video/video.flv", controls);
addChild(player);
Get a skin

var player:SomaVideoPlayer = new SomaVideoPlayer("video/video.flv");
addChild(player);
var skin:SomaVideoMuteSkin = player.controls.getControl(SomaVideoMuteSkin) as SomaVideoMuteSkin;
skin.backgroundColor = 0xFF0000;
skin.backgroundAlpha = .5;
See the SomaVideoControls documentation to add or create your own controls.
