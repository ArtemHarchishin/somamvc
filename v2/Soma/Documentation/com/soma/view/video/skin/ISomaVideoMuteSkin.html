Usage:
Interface meant to implemented by a DisplayObject class (such as Sprite) to skin a volume mute button skin and be added to the SomaVideoControls (somaVideoPlayerInstance.controls). See the SomaVideoMuteSkin documentation to create your own volume mute skin and the SomaVideoControls documentation to add or create your own controls.
