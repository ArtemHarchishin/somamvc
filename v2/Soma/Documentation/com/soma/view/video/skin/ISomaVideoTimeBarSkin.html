Usage:
Interface meant to implemented by a DisplayObject class (such as Sprite) to skin a buffer/preloading/time bar skin and be added to the SomaVideoControls (somaVideoPlayerInstance.controls). See the SomaVideoTimeBarSkin documentation to create your own buffer/preloading/time skin and the SomaVideoControls documentation to add or create your own controls.
