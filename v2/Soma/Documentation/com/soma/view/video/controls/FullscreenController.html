Usage:
FullscreenController is part of the default video skin system of the SomaVideoPlayer. It is a controller class that is sending data to its skin class (ISomaVideoFullscreenSkin).

Internally used by the SomaVideoControls class, it is not meant to be instantiated by the user. See the SomaVideoControls documentation to add or create your own controls.
